Hodge Podge Docs

Version 1.0

Core Functionality

Hodge Podge has a couple of features that are build into the core bot, other then these, there are many other features that have been attached in modules. Note that core functionality is at the bottom of the call chain and will be activated only if no other module has already been activated by the message.

Triggering

You can trigger hodge podge commands anywhere in a sentence by writing !hp. For example !hp roll a d5 You can also do Hodge Podge or Hodgepodge case insensitive. You can also mention hodge podge @HodgePodge roll a d5

Help

You can ask hodgepodge for some help, he'll just link you back to this page. You can trigger hodgepodge to send you the link via

  • Hodge Podge help
  • Hodge Podge docs
  • Hodge Podge show documentation
  • Hodge Podge how do i...

Personality
