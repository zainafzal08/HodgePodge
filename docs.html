Hodge Podge Docs

Version 1.0

Core Functionality

Hodge Podge has a couple of features that are build into the core bot, other then these, there are many other features that have been attached in modules. Note that core functionality is at the bottom of the call chain and will be activated only if no other module has already been activated by the message.

Triggering

You can trigger hodge podge commands anywhere in a sentence by writing !hp. For example !hp roll a d5 You can also do Hodge Podge or Hodgepodge case insensitive. You can also mention hodge podge @HodgePodge roll a d5 Hodge Podge help HodgePodge help hodgepodge help !hp help @HodgePodge help

Help

You can ask hodgepodge for some help, he'll just link you back to this page. You can trigger hodgepodge to send you the link via Hodge Podge help Hodge Podge docs Hodge Podge show documentation Hodge Podge how do i...

Enviornment Variables

Many modules will use some variables to set things like standard responses or allowed ranges. See the dice module for a example. These variables are server wide and can be updated by doing Hodge Podge set X to Y Got it! X must be a variable name and as such is 1 word and snake cake i.e critical_hit_msg. The first letter of the varibale must not be a underscore, the allowed characters for a variable is any character a-z and A-Z + the underscore.

You can reveal a server variable via the reveal command

Hodge Podge reveal X X = Y

Dice Module

Basic Use

You can simply say Hodge Podge Roll a dx where x is any number 2 to 10000. To roll multiple dice specify a number of dice to roll i.e Hodge Podge roll 5 d10 . Because hodge podge just looks for the phrase "roll" followed by some numbers you can make your requests more dramatic. Hodge Podge roll a d10 Hodge Podge roll 5 d6 Hodge Podge my dude roll me a fat d8 Hodge Podge please roll 7 of those hot hot d18 Please note that Hodge podge will display the individual dice rolls if you roll up to 20 dice. Any more and Hodge podge just reports the total. Hodge Podge roll 3 d2 I got 4! The breakdown was (1,2,1) Hodge Podge roll 25 d2 I got 28 total!

Modifiers

You can also apply modifiers by specifying a plus or minus after your roll. Hodge Podge roll a d10 +3 I got 8! 5[+3] Hodge Podge roll a d10 -3 I got 2! 5[-3] If you want to apply a modifier to a multi dice roll you have to specify if you want the modifier to apply to the total... Hodge Podge roll 2 d10 +3 I got 13! The breakdown was (5,5)[+3] Or if you want it to apply to every dice roll Hodge Podge roll 2 d10 ++3 I got 16! The breakdown was (5[+3],5[+3])

Rerolling

Hodge podge will remember the last roll in every channel and will reroll it when asked. you simply need to say Hodge podge reroll , hodge podge will ignore all text after the reroll word Hodge Podge roll 2 d10 ++3 I got 16! The breakdown was (5[+3],5[+3]) Hodge Podge reroll that I got 11! The breakdown was (2[+3],3[+3])

This is actually just using a server variable called last_roll to remember the last thing you rolled. You can manually override it and look at it but do so at your own risk. malforming it will cause "reroll" to fail.

When hodge podge rolls a critical miss or a critical hit for a single dice he will say "oof" or "nice!". Utility Module

Decisions

Hodge Podge can flip a coin or pick a random person in the chat. Note when picking a random person hodge podge will tag that person. This is useful for when you need to pick a random character to hex or hit in a battle.

Hodge Podge flip a coin It landed on heads! Hodge Podge pick someone I am gonna point at @Travis!

By default hodge podge treats all people in a channel equally no matter if they are online at the moment or not.

Conversions

To convert between feet and meters simply do

Hodge Podge whats 14 feet in meters It's 4.2672 meters! Hodge Podge whats 4.2672 meters in feet It's 14 feet!

To convert between celcius and fahrenheit simply do

Hodge Podge whats 14F in C It's -10C! Hodge Podge whats -10C in F It's 14F!

Scripting

Ok so i am a bit crazy and look for any excuse to build a compiler from scratch, So Hodge Podge comes with a very basic scripting engine which runs HodgeCode. If you arn't a programmer this may seem useless but keep reading, it has some usful features relevant to everyone!

Math

The scripting engine will evaluate any mathematical expression involving standard BODMAS

Hodge Podge do 5+4 9 !hp do 2*2 - 2 + 1 3

Dice

You can slip dice rolls into your mathematical equations, hodge podge will roll a dice and replace the result into your equation. !hp do d20+4-2*2 2

Variables

Hodge Code can also access your server variables which lets you do some cool stuff like save variables relevant to play !hp set orc_bonus_damage to 8 Got it! !hp do d6+orc_bonus_damage 10
